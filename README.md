# 银河恶魔城DEMO


## 项目简介  
- 一款基于Unity开发的银河恶魔城游戏
- 开发周期：2周（个人独立开发）  
- 技术栈：Unity 2021、C#、UGUI、FSM（状态机）

## 核心功能  
1. 玩家控制系统：WASD移动，跳跃，攻击，滑墙，冲刺  
2. AI敌人： 状态机控制。实现功能：巡逻，探测玩家，跟随玩家
3. 关卡设计：tilemap搭建地图
4. 战斗系统：攻击，弹反，血条
5. UI：血条

## 操作说明  
- 移动：WASD  
- 跳跃：空格  
- 反击：Q
- 冲刺：shift
